# SafeDrive Vision 🚗

AI-powered Driver Monitoring System built with Python.

## Features
- Eye closure / drowsiness detection
- Yawn detection
- Look-away detection
- Phone usage detection
- Real-time focus score
- Voice alerts
- SQLite logging
- Premium dashboard UI
- AI camera HUD overlay

## Tech Stack
- Python
- OpenCV
- Dlib
- PyTorch (YOLOv5)
- PySide6
- SQLite
- Pygame

## How to Run
```bash
python SafeDriveVisionFinal.py
```
